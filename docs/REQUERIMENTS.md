# Especificaciones del Proyecto Integrador
**Curso:** Álgebra Lineal
**Proyecto:** Análisis y Resolución de Ecuaciones y Sistemas Lineales con Python

Desarrollar una aplicación en Python que permita resolver ecuaciones lineales y sistemas de ecuaciones 2x2 mediante algoritmos estructurados y lógica matemática.

## Objetivo General

Desarrollar un programa en Python que:

1. Verifique si una ecuación es lineal.
2. Resuelva ecuaciones de primer grado.
3. Resuelva un sistema de dos ecuaciones lineales con dos incógnitas.
4. Clasifique el sistema según el valor del determinante.
5. Muestre el procedimiento matemático y la interpretación del resultado.

## Componentes del Programa

El programa deberá incluir un menú interactivo con las siguientes opciones:

1. Verificar si una ecuación es lineal.
2. Resolver ecuación lineal simple.
3. Resolver sistema 2x2.
4. Salir.

## Requisitos Técnicos Obligatorios

- Uso de funciones.
- Manejo de errores con try/except.
- Validación de entradas.
- Código comentado.
- Salida clara y comprensible.
- Mostrar el procedimiento matemático.

## Especificaciones Adicionales

El programa también deberá:

- Mostrar paso a paso el cálculo del determinante.
- Indicar qué tipo de sistema es.
- Explicar el resultado en términos matemáticos.
- Generar al menos 3 sistemas de prueba dentro del código.
